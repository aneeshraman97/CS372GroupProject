Details

Doc-Dash is a free to use website where you can send a file to friends, co-workers, and family. It's fast and easy, because no sign-up is required! If you want to keep your document secure, add a pin code and send that to your buddy. Don't worry about the file being stolen either, we delete it after it's downloaded.
